Introduction

  • ILP is an extension of LP that solves LP models that require integer solutions. Integer programming deals with the special linear programming situation in which the decision variables are constrained to take non-negative integer values.
  • Some business situations require integer solutions in solving LP models that involve Yes/No decisions. We use binary variables (take only the values 0 and 1) to solve such LP models.

Learning Objectives/Outcomes

  • Understand the basic ideas of integer linear programming and the main techniques for formulating ILP models
  • Solve optimisation models in which certain decision variables take only integer values
  • Discuss how the use of integer variables allows us to build more accurate decision models for a number of business situations
  • Understand how to set-up integer programming business models in a spreadsheet and solve them using Excelâ€™s Solver.
