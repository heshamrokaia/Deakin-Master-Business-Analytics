Introduction

  • In solving an LP problem we assume that values of all model coefficients are known with certainty, but such certainty rarely exists.
  • Sensitivity analysis helps answer questions about how sensitive the optimal solution is to changes in various coefficients in a model.
  • The output generated from linear programming packages (including Solver) provides useful “what if” analysis. For example, once an optimal solution has been found, what would be the effect if just a little extra of one or other of the resources could be found? Or if the profit of one of the possible items is increased by just a little bit?
