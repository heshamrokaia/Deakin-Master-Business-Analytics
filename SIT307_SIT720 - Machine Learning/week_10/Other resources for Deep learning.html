Other resources for Deep learning

This was a very brief introduction to deep learning. This a growing field that will greatly affect our future. For those of you who are interested, here are some useful resources:

Books

  • Deep learning textbook (Goodfellow et. al., 2016)
  • Deep learning: an overview (Jurgen, 2015)
  • Deep learning (Michael Nielsen, 2016)
  • A statistical view of deep learning (Shakir Mohamed, 2015)

Courses/Tutorials

  • Hugo Youtube channel
  • Colah’s blog: very nice visualizations for easier and better understanding.
  • Karpathy’s blog: deep learning in web browsers.

Tools

  • Keras
  • Theano (Montreal – Bengio et. al.)
  • Lasange, Keras
  • Caffe(Berkeley)
  • Torch(Facebook)
  • Deep Scalable Sparse Tensor Network Engine (DSSTNE)(Amazon)
  • MatConvNet(Oxford, MATLAB)
