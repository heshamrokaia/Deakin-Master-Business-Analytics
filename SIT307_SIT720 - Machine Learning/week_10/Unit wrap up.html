Unit wrap up

Congratulations! You have completed week 10  as well as this unit on machine learning.

In this unit, you have learned various unsupervised and supervised machine learning algorithms and applied them to solve practical problems using Python programming. Besides the mathematical formulation that we studied machine learning algorithms, you have also gained substantial skills in how to use machine learning algorithms to solve real-world problems. You have also explored generalisation of machine learning models and the effect of data distribution on machine learning models. I hope you have enjoyed the journey through the unit. We hope that you have enjoyed the unit and that you will find the learning useful in future.
