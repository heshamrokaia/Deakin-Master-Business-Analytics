Weekly wrap up

Congratulations, you have completed Week 3!

This week was about understanding clustering. We have looked at several clustering algorithms, how they work as well as their limitations. We have also looked at how the algorithms could be evaluated using various benchmarks. Next, we implemented several Python programs to explore these concepts in practice.

Next week, we will continue to explore several related concepts. These include:

  • dimensionality of data
  • Principle Component Analysis (PCA)
  • eigenvalues and eigenvectors
  • Singular Value Decomposition (SVD).

We look forward to exploring these ideas with you next week.
