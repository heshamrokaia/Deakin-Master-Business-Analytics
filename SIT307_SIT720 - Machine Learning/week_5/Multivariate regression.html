Multivariate regression

Now, lets look at a multivariate example. In this example, we look at the X1 vs Y1 (feature vs class/target).

We can now build our regression model. We proceed by dividing this original dataset into two parts:

  • Training dataset: containing 67% of original data
  • Testing dataset: containing  33% of original data

Why do we do this? So that we can effectively judge our model. We build our model using training data. Then we test its performance on the unseen(by the model) test data.

For now, lets manually create the train/test split using 67% training data and 33% testing data.

Code example# 1

from sklearn.model_selection import train_test_split
X_train, X_test, y_train, y_test = train_test_split(df.iloc[:,0:-1], df.iloc[:,-1], test_size=0.33, random_state=42)
print("Train shape: ",X_train.shape)
print("Test shape: ",,X_test.shape)
