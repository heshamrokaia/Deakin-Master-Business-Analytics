AdaBoost:

AdaBoost, which stands for Adaptive Boosting, is a machine-learning algorithm for classification problems. It works by combining weak classifiers to create a strong classifier. The algorithm has the following steps:

Inputs:

  • X: dataset of features
  • y: vector of corresponding labels (+1 or -1)
  • T: number of iterations (i.e., number of weak classifiers to train)

Outputs:

  • List of weak classifiers, each with an associated weight
