Outlier Detection using PyOD Library

 

At the starting of this week, we learned about anomalies or outliers.  An outlier is any data point that differs greatly from the rest of the observations in a dataset. In this section, we will demonstrate a python based toolkit PyOD for outlier detection.

 

PyOD is a comprehensive and scalable Python toolkit for detecting outlying objects in multivariate data. This exciting yet challenging field is commonly referred to as Outlier Detection or Anomaly Detection. 

 

 

Installing PyOD 

pip install pyod
pip install --upgrade pyod  # # to make sure that the latest version is installed!

Alternatively, you could clone and run setup.py file:

git clone https://github.com/yzhao062/pyod.git
cd pyod
pip install .

*We have already installed the PyOD in your VM. So, you don’t need to take any action if you run from VM.
