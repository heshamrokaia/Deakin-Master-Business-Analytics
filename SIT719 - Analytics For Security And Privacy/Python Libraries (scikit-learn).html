scikit-learn

Scikit-learn is an open source machine learning library that supports supervised and unsupervised learning. It also provides various tools for model fitting, data preprocessing, model selection and evaluation, and many other utilities. In this unit, we will heavily use this powerful machine learning library to demonstrate the applicability of data analytics capabilities towards solving security issues.

It includes submodules for such models as:

  • Classification: SVM, nearest neighbors, random forest, logistic regression, etc.
  • Regression: Lasso, ridge regression, etc.
  • Clustering: k-means, spectral clustering, etc.
  • Dimensionality reduction: PCA, feature selection, matrix factorization, etc.
  • Model selection: Grid search, cross-validation, metrics
  • Preprocessing: Feature extraction, normalization

Example:

Load a dataset using scikit learn dataset module. To load in the data, you first import the module datasets from sklearn. Then, you can use the load_iris() method from datasets to load in the data:

# Import `datasets` from `sklearn`
from sklearn import datasets
# Load in the `iris` data
iris_dataset = datasets.load_iris()
# Print the `iris` data
print(iris_dataset)

The sklearn.datasets module includes utilities to load datasets, including methods to load and fetch popular reference datasets. It also features some artificial data generators. The list can be accessed from the following link:

https://scikit-learn.org/stable/modules/classes.html#module-sklearn.datasets
