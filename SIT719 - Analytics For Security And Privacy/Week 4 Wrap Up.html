Week 4 Wrap Up

Well done! You have reached the end of week 4. This week, at the start, we focused on the fundamental concepts of supervised machine learning.  Next, we introduced different performance measures. The powerful machine learning and data mining tool WEKA is also introduced in this week. We have then utilized the week to solve one cyber security related problem.  Here, we have used widely used NSL-KDD dataset for network intrusion detection. WEKA based classification tasks are explored to detect attack instances.  

This week helped you towards solving a challenging security problem related to the intrusion detection system. You have learned how to used state-of-the-art classification algorithms using a powerful tool like WEKA.  

Hope you enjoyed this week.
