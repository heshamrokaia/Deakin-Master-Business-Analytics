Week 5 Wrap Up

 

This week, we explored the supervised learning algorithms and demonstrated some use-cases using python libraries.

We have learned the following concepts and skills:

1. Fundamental knowledge on Naive Bayes, Decision Tree, Random Forest, Support Vector Machine and Neural Network.

2. Python scripting using scikit-learn to implement IDS for NSL-KDD cup dataset and UNSW NB15 dataset.

3. Discussion of results using benchmark performance matrices.

This week helped the students towards achieving the ULOs related to 1, 3, and 5.
