Weekly Wrap Up

 

Well done! You have reached the end of week 2. This week, you have learned an important analytical skill- Python scripting!

Python is a powerful programming language to solve challenging and emerging analytical problems using its extended machine learning and AI-based libraries.  We will be using this tool throughout the semester to identify network attacks and intrusions.  

We started the week by discussing what is python and how it is important for data science tasks. Next, we discussed important python components, like data types, modules, and libraries. The major focus of this week is to understand and become familiar with the scientific Python libraries like Pandas, Numpy and Scikit learn.  We have practiced a good number of sample examples to understand the scripting basics.
