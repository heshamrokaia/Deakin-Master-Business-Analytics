Welcome to Week 5

This week, we will continue to explore the supervised learning algorithms and demonstrate some use-cases.

The focus of this week is to review some of the widely used powerful supervised learning algorithms. Specifically, we will explore Naive Bayes, Decision Tree, Random Forest, Support Vector Machine and Neural Network. Next, we will implement these algorithms using python scikit-learn. We will demonstrate two examples using NSL KDD dataset and UNSW NB15 dataset.

Once a student completes this week successfully, it will significantly help them by developing the advanced knowledge of supervised machine learning for security analysis, be familiar with python-based scripting for classification of attack data for large datasets. This week will help the students towards achieving the ULOs related to 1, 3, and 5.
