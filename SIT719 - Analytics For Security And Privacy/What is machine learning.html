What is Machine Learning?

 

Humans learn from past experience. On the other hand, machines follow instructions given by humans. But what if humans can train the machines to learn from past data and do what humans can do as well as act must faster? Yes, that is machine learning. However, it is more than learning. It is also about understanding, reasoning, and predicting.

 “Machine Learning is the science of getting computer algorithms to learn and act like humans do, and improve their learning over time in autonomous fashion, by feeding those data and information in the form of observations and real-world interactions.”

 

Some Machine Learning Definitions

Yoshua Bengio from the Université de Montréal, who is one of deep learning’s “founding fathers” describes machine learning as below:

Machine learning research is part of research on artificial intelligence, seeking to provide knowledge to computers through data, observations and interacting with the world. That acquired knowledge allows computers to correctly generalize to a new setting.

 So the fundamental principle of machine learning relies on the learning process. 

 According to the Nvidia website:

Machine Learning at its most basic is the practice of using algorithms to parse data, learn from it, and then make a determination or prediction about something in the world. So rather than hand-coding software routines with a specific set of instructions to accomplish a particular task, the machine is “trained” using large amounts of data and algorithms that give it the ability to learn how to perform the task.

 Researchers from the Machine Learning Department of the Carnegie Mellon University defined the central question that Machine Learning studies. According to them, the field of Machine Learning seeks to answer the following question:

“How can we build computer systems that automatically improve with experience, and what are the fundamental laws that govern all learning processes?”
