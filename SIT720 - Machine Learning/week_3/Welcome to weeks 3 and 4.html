Welcome to unsupervised machine learning

In the previous two weeks you started to explore the basics of machine learning, coded a number of Python programs to help you understand its versatility in implementing machine learning principles, and revisited concepts in linear algebra which are fundamental to this topic.

In the next two weeks we will look at two aspects of unsupervised machine learning on two forms of unsupervised learning:

  • Clustering
  • Dimensionality reduction

Besides the theoretical understanding of these approaches, we'll learn how we can use them in practical applications. Clustering can find patterns in large data sets when the data is not labelled and dimensionality reduction helps us when the number of features, that describe the difference between data, becomes too large to manage.

 In the programming part of this unit, we'll learn how to use Python packages to create unsupervised models, reduce dimensionality of the data and how to visualise high dimensional data. 

When we talk about dimensions we are talking about the features or attributes or characteristics that distinguish on.

Learning Goals

By the end of week 4 you will be able to:

  • use clustering for revealing patterns from unlabelled data
  • learn techniques to reduce dimensionality
  • apply suitable clustering/dimensionality reduction techniques to perform unsupervised learning of data in a real-world scenario.
