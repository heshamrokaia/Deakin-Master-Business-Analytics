Welcome to week 5

Welcome back to week 5, Fundamentals of Supervised Learning. In the next two weeks we'll focus on some key concepts important for supervised learning. These include model complexity, model assessment, as well as model selection. The theoretical understanding of these concepts will help you to asses any train model and select the best model from a set of models. We've also learned how to use Python packages for assessing and selecting the model.

In the previous two weeks you investigated two forms of unsupervised learning: Clustering and Dimensionality reduction. In addition to a theoretical understanding of these approaches, you have also learned how to use them in practical applications.

Learning Goals

By the end of weeks 5 and 6 you will be able to:

  • differentiate supervised learning from unsupervised learning
  • estimate the performance of different supervised learning models
  • implement model selection and compute relevant evaluation measures
