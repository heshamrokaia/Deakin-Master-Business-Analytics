Support vector regression

Similar to the support vector classification, we have an algorithm for support vector regression.

It is assumed that the function we learn is within ϵ{"version":"1.1","math":"\(\epsilon\)"} deviation of the target values (ϵ{"version":"1.1","math":"\(\epsilon\)"} 

- tube). We also want the function to be as flat as possible.

In Support Vector Regression (SVR), the classifier still defines a margin (it looks like a tube in the image) and as long as the data points are in the tube they are clear and fine. As soon as a data point goes outside the tube or margin, we call the point deviation. This is similar to the concept in almost separable data points in previous lessons.

So again we define a ζ{"version":"1.1","math":"\(\zeta\)"} 

and try to look for a good level of complexity and also accurate results. Remember, by enforcing very small numbers of deviations or misclassified points, you will create a margin, or in this case a tube, which most of the points fall inside.

Similar to other problems in machine learning, by having this complicated model you have the risk of over-fitting.
